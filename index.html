스캔본 OCR

OCR은 이미지에서 글자를 추정하는 방식이라 100% 정확하지 않습니다. 표·작은 글씨·노이즈가 많은 스캔본은 오인식이 생길 수 있습니다.

LLM 보정 번역 (선택)

API 키는 이 브라우저(localStorage)에만 저장되며 외부로 전송되지 않습니다. 번역 요청은 사용자가 입력한 키로 해당 제공자에게 직접 호출됩니다.
